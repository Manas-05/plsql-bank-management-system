# Bank Management System (PL/SQL)

A backend banking database system developed using Oracle PL/SQL.

## Features
- Customer creation using stored procedures
- Account management
- Deposit and withdrawal operations
- Balance validation using business rules
- Transaction logging
- Exception handling with RAISE_APPLICATION_ERROR

## Technologies Used
- Oracle SQL
- PL/SQL
- Stored Procedures
- Sequences
- Foreign Keys
- Transaction Control (COMMIT)

## Testing
Tested in Oracle Live SQL environment with:
- Successful deposit operations
- Successful withdrawals
- Insufficient balance validation
- Transaction history tracking
